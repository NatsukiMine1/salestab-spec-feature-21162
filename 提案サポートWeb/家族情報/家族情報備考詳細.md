## 画面項目

1. 画面名
    - [x] 「家族情報の備考」が表示される。
2. 閉じるボタン
    - [x] 押下すると、[閉じるボタンを押下](#閉じるボタンを押下)を実行する。
3. 編集ボタン
    - [x] 押下すると、[編集ボタンを押下](#編集ボタンを押下)を実行する。
4. 備考
    - [x] 家族情報備考の内容が表示される。
    - [x] 表示領域を超える文字数がある場合は、スクロールして全文を確認できる。
    - [x] 登録されていない場合は何も表示されない。
    - [x] 備考の下に注釈が表示される。

## イベント
